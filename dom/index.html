What is the DOM?

  • Document Object Model
  • Programming interface for web pages
  • Represents the document as nodes and objects
  • Allows programs to change document structure, style, and content.

DOM Interface

  • Document - the root document object
  • Node - element, text, or attribute
  • Element - node of type Element
  • NodeList - array of elements
  • document.querySelector()
  • document.querySelectorAll()
  • document.createElement()
  • parentNode.appendChild(node)
  • element.innerHTML
  • element.style.left
  • element.setAttribute()
  • element.getAttribute()
  • element.addEventListener()
